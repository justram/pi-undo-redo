## [0.2.0] - 2026-02-04

- Added `/undo-redo-clear-cache` command to clear the undo/redo cache for the current session.

### Changed
- Updated default undo/redo shortcuts to `ctrl+shift+z` and `ctrl+shift+y`.
- Documented undo/redo reset behavior when resuming sessions.
